# Habit Tracker

## Besoin
Je veux une page "Habit Tracker" dans mon application personnelle d’organisation.

## Objectif utilisateur
Pouvoir suivre facilement mes habitudes quotidiennes dans une vue claire et visuelle.

## Résultat attendu
Je veux voir :
- une liste de mes habitudes
- une grille avec une case par jour
- la possibilité de cocher ou décocher une habitude pour un jour
- un pourcentage de réussite par jour
- des visualisations simples, par exemple un pie chart
- une expérience fluide et simple

## MVP
- page dédiée Habit Tracker
- liste d’habitudes
- grille journalière
- cases cochables
- sauvegarde des données
- % de réussite du jour

## Améliorations futures
- stats hebdomadaires
- stats mensuelles
- pie charts
- streaks
- filtres
- catégories d’habitudes

## Cas limites
- aucune habitude créée
- nouvelle habitude créée en milieu de mois
- suppression d’une habitude
- journée sans données
- affichage responsive
